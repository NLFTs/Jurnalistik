# #Apa Itu Laravel?

Laravel adalah sebuah framework PHP yang dirancang untuk memudahkan pengembangan aplikasi web dengan sintaks yang elegan dan ekspresif. Laravel mengikuti pola arsitektur moderen yaitu Model-View-Controller (MVC), yang memisahkan logika bisnis dari presentasi data.

## #Kenapa Harus Pakai Laravel?

1. **Super Simple**: Laravel memiliki pola arsitektur yang cukup mudah untuk di bangun. sehingga cocok untuk web web UMKM kecil dan juga perusahaan besar. dengan waktu pengerjaan yang lebih cepat
2. **Kontrol penuh Terhadap Database**: Laravel juga menyediakan Sistem yang di namakan dengan Migration, ini memungkinkan pengembang untuk membuat database langsung didalam framework
3. **Frontend Support**: Laravel juga menyediakan starterkit berupa Framwork moderen yang bisa di gunakan di dalam Laravel tanpa perlu intergasi dengan API. Ruanglingkup starterkit ini sendiri mencakup Framework-Framework Moderen Seperti React dan Vue
